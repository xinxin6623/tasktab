# TaskBoard 指导文档包

把整个文件夹丢进工作目录,对 Claude Code 说:

> 读 taskboard-docs/ 下全部文档,按 02-实现步骤.md 的里程碑顺序实现 TaskBoard,每完成一个里程碑跑完验收项再继续。

阅读顺序:

1. `01-大白话说明书.md` — 产品意图与设计理由(给人看,AI 也应读以对齐目标)
2. `02-实现步骤.md` — 执行计划:数据契约、技术栈、M1–M5 里程碑与验收标准(主文档)
3. `03-SKILL创建规则.md` — progress-tracker skill 规范与完整草稿(M5 阶段使用)

冲突裁决:数据契约以 02 第 1.1/1.2 节为唯一权威;产品行为以 01 为准;skill 细节以 03 为准。
